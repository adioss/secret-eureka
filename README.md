## Run Locally

Clone the project

```bash
  git clone git@github.com:adioss/unconcealment.git
```

Go to the project directory

```bash
  cd unconcealment
```

Install poetry and install dependencies

```bash
  poetry self update
  poetry install
```

Start

```bash
  poetry run python unconcealment/main.py 
```
